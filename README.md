# XAO Attendance Tracker

This is a static HTML/CSS/JS page intended to be hosted on GitHub Pages.

## Deploy on GitHub Pages (quick)

1. Create a new GitHub repository (e.g., `xao-attendance-tracker`)
2. Upload the files from this folder (`index.html` and `.nojekyll`)
3. In the repo: **Settings → Pages**
4. **Source:** Deploy from a branch → **Branch:** `main` → **Folder:** `/ (root)` → Save
5. Your site will be available at:
   `https://<your-username>.github.io/<repo-name>/`

## Embed in MemberClicks

```html
<iframe src="https://<your-username>.github.io/<repo-name>/" width="100%" height="1400" style="border:0;" loading="lazy"></iframe>
```
